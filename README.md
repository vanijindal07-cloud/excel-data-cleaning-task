# Excel Data Cleaning Task

This repository contains Task 1 of the Data Analyst Internship.

## Dataset
Netflix Movies and TV Shows dataset.

## Work Done
- Identified and handled missing values
- Removed duplicate titles after creating a raw backup
- Cleaned text fields using TRIM
- Standardized date format
- Created a Cleaned_Data sheet
- Added Data_Quality_Notes column
- Exported cleaned data as CSV

## Files Included
- Raw_Data.xlsx
- Cleaned_dataset.xlsx
- cleaned_dataset.csv
